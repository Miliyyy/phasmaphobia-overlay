# Phasmo Journal

A real-time overlay for Phasmophobia that sits on top of your game.

## Features
- Evidence tracker — click to mark found/ruled out, auto-filters ghost list
- Ghost database — all 24 ghosts with strengths, weaknesses, tells, and speed
- BPM speed tool — tap to ghost footsteps, calculates m/s and matches ghost type
- Map guide — all 12 maps with ghost rooms, hiding spots, and strategy tips
- Hunt timers — hunt duration, smudge countdown (90s), team sanity tracker
- Investigation journal — save rounds with one click, search and filter history
- Global hotkeys — F1/F9/F10 work even while playing (no alt-tab needed)

## Usage
1. Run Phasmophobia in Borderless Windowed mode
2. Launch PhasmoJournal.exe
3. Drag the overlay anywhere on screen

## Build from source
Requires Python 3.10+
Double-click BUILD.bat

## Hotkeys
| Key | Action |
|-----|--------|
| F1 | Start/stop hunt timer |
| F9 | Start smudge timer |
| F10 | Tap ghost footstep (BPM) |
| F11 | Reset BPM |
| Alt+1–7 | Toggle evidence |
